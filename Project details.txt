Description:
Se busca un desarrollador con experiencia comprobable en FlutterFlow y Firebase para crear un Producto Mínimo Viable (MVP) de una aplicación móvil de servicios a domicilio. El proyecto requiere un enfoque en la funcionalidad principal y una base sólida para futuras expansiones. Las funcionalidades clave para la Fase 1 incluyen: Registro e inicio de sesión para usuarios (clientes y técnicos). Capacidad para que los clientes creen solicitudes de servicio detalladas. Funcionalidad para subir fotos relacionadas con el problema o servicio requerido. Integración de geolocalización utilizando Google Maps para mostrar la ubicación del cliente. Un sistema básico de estimación de costos basado en reglas simples. Desarrollo de un panel de administración para la gestión eficiente de todas las solicitudes de servicio. Implementación de un botón de contacto (WhatsApp o un chat simple dentro de la app). Estructuración de la base de datos en Firebase, incluyendo autenticación (Auth) y almacenamiento de datos (Firestore). Requisitos del perfil del freelancer: Experiencia previa demostrable en el desarrollo y publicación de aplicaciones móviles. Dominio en la integración de la API de Google Maps. Amplio conocimiento y manejo de Firebase (Auth, Firestore). Compromiso con la entrega del proyecto en un formato editable y con acceso completo a todo el código fuente y recursos. Habilidades de comunicación claras y capacidad para cumplir con los cronogramas establecidos. Se espera recibir propuestas con un presupuesto fijo y un cronograma estimado para la finalización de esta primera fase del proyecto.
